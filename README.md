# Branch de Continuidade do Projeto
---
A branch **`main`** contém exatamente a versão do projeto que foi desenvolvida e entregue dentro do prazo da atividade.
Esta branch (**`renan`**) foi criada para que eu possa continuar evoluindo o projeto como forma de estudo e aprimoramento pessoal. As alterações feitas aqui — como melhorias, refatorações e novas funcionalidades — não modificam a versão que foi oficialmente entregue.
---
Assim, a branch principal **`main`** permanece como registro fiel da entrega original, enquanto esta branch serve apenas para continuidade dos estudos.
